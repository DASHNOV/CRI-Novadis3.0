# Novadis Projet Global

Ce référentiel contient le code source complet pour l'application Novadis, comprenant à la fois le backend et le frontend.

## 📂 Structure du projet

- **backend/** : Contient l'API et la logique serveur (.NET / ASP.NET Core).
- **frontend/** : Contient l'application mobile (Flutter).

## 🚀 Pour commencer

### Backend
Veuillez vous référer au dossier `backend/` pour les instructions spécifiques concernant l'API.

### Frontend
Veuillez vous référer au dossier `frontend/` pour les instructions d'installation et de lancement de l'application mobile.
Consultez le [README du frontend](frontend/README.md) pour plus de détails.
